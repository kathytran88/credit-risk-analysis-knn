Built 3 machine learning models to classify "Risk" or "No Risk" of credit loaning customers for banks<br>
- K nearest neighbors: acc = 0.72<br>
- Decision Tree: acc = 0.73733333<br>
- Random Forest: acc = 0.7605<br>
